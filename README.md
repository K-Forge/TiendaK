# 🛒 Tienda-K

**Tienda-K** es una aplicación web desarrollada para la tienda de productos de la **Fundación Universitaria Konrad Lorenz**. Permite a los usuarios realizar compras en línea con opción de recogida o entrega a domicilio, y ofrece un sistema de gestión de inventario para empleados y administradores.

## 🚀 Funcionalidades principales (planeadas)

- Navegación por catálogo de productos universitarios
- Compra en línea con opción de recogida o entrega
- Registro e inicio de sesión para usuarios
- Gestión de inventario por parte de empleados
- Administración de productos en stock y nuevos ingresos al almacén

## 🛠 Tecnologías

- **Backend:** Java
- **Base de datos:** SQL (motor por definir)
- **Frontend:** Por definir

> ⚠️ Este proyecto se encuentra en etapas tempranas de desarrollo. Las tecnologías y funcionalidades pueden cambiar conforme avance el desarrollo.
